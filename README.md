# Contact-Manager-in-C
 // Description  A Mini Simple Contact Management System developed in C.

The application allows users to:

- Add contacts
- View all contacts
- Search contacts
- Delete contacts

All contact information is stored in a text file using C file handling functions.

---

## Features

- Add new contacts
- Display saved contacts
- Search by name
- Delete contacts
- Persistent file storage

---

## Technologies

- C
- File Handling
- Structures
- String Manipulation

---

## File Structure

contacts.txt

Format:

Name|Email|Phone

Example:

John Doe|john@gmail.com|0612345678
